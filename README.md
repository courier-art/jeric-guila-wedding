# Guila & Jeric Wedding Website

A modern minimalist wedding website with elegant black and white design.

## Features
- 6 responsive pages
- Countdown timer to wedding date
- Interactive RSVP form with Google Sheets integration
- Wedding party display with 10 sponsors, 10 best men, and 10 bridesmaids
- Message board for guests
- Event details with maps
- Dress code information
- Parking instructions

## Setup for GitHub Pages

1. Create a new repository on GitHub
2. Upload all files to the repository
3. Go to Settings > Pages
4. Under "Source", select "Deploy from a branch"
5. Select the `main` branch and `/root` folder
6. Click Save - your site will be live at: `https://[username].github.io/[repository-name]/`

## Image Placeholders

The website uses placeholder images named image01.jpg through image34.jpg. Replace these with your actual wedding photos.

## RSVP Form Setup

1. Create a Google Sheet
2. Tools > Script Editor
3. Paste the Apps Script code
4. Deploy as Web App
5. Update the scriptURL in rsvp.html with your deployment URL

## Customization

Edit the HTML files to:
- Update names, dates, and locations
- Replace placeholder images
- Modify color scheme if needed
- Add your Google Apps Script URL for RSVP functionality
